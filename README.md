# 这是我web开发课程的平时作业代码仓。

作业要求见：[https://gitee.com/bigbug55/webdesign](https://gitee.com/bigbug55/webdesign)

### 项目任务
* **Task1-7**：前端项目，均完成vercel托管。
* **Task8**：基于火山引擎的脑筋急转弯项目。
* **Quiz系统部署Ubuntu**：
    * **基于Vue的管理端前端**——实现管理员对题目和普通用户增删改查
    * **用户测试前端**——实现普通用户注册登录和做测试（Quiz）
    * **基于Spring Boot与JWT的后端**——实现多级权限校验与在线测试业务
    * **部署情况**：我将系统完整部署到MacBook的Ubuntu虚拟机并完成验收：利用Nginx托管多端静态资源并实现后端反向代理，统一处理跨域限制并完成数据库迁移部署。
* **QuizReact**：使用React重写Quiz系统。

### 本课程主要的内容
* **前端**：html/css/js + Vue + React
* **前端组件框架**：ElementUI + AntDesign
* **后端**：Java Spring / Java Web / Mysql
* **项目**：前后端全栈练习项目
